# Plot Control Chart

Creates a control chart visualization showing data points, control
limits, and any rule violations.

## Usage

``` r
# S3 method for class 'measure_control_chart'
autoplot(object, ...)
```

## Arguments

- object:

  A
  [measure_control_chart](https://jameshwade.github.io/measure/dev/reference/measure_control_chart.md)
  object.

- ...:

  Additional arguments (unused).

## Value

A ggplot object showing the control chart.
