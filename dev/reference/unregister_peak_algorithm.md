# Unregister a Peak Detection Algorithm

Removes a peak detection algorithm from the registry.

## Usage

``` r
unregister_peak_algorithm(name)
```

## Arguments

- name:

  Algorithm name to remove.

## Value

Invisible `TRUE` if removed, `FALSE` if not found.
